basic

선언

s = {"a","b"}

원소추출

- 일반 인덱스로는 못한다.

s[0]
TypeError: 'set' object is not subscriptable

- 딱히 하는 방법이 없음

- 무엇보다 이걸 하는 의미가 없다.(원소에 접근해서 하는거 자체가 무의미한 행위)

원소 추가

s.add("c")
s
{'a', 'b', 'c'}

원소 삭제

s.remove("c")
s
{'a', 'b'}

연산

in

"a" in s
True

합,교,차집합

set1 = {"a","b","c"}
set2 = {"b","c","d"}

- 합집합

set1|set2
{'a', 'b', 'c', 'd'}
set1.union(set2)
{'a', 'b', 'c', 'd'}

- 교집합

set1 & set2
{'b', 'c'}
set1.intersection(set2)
{'b', 'c'}

- 차집합

set2-set1
{'d'}

부분집합

set3 = set2|set1
set1<set3 
True
set2<set3
True

for문과 set

for i in set1|set2:
    print(i)
b
d
c
a

Advanced

- set은 중복을 허용하지 않음

lst = [1,2,1,1,3,4,5]
set(lst)
{1, 2, 3, 4, 5}

- 다음의 텍스트가 각각 몇번씩 사용되었는지 빈도를 구해라

txt = 'asdkflkjahsdlkjfhlaksglkjdhflkgjhlskdfjhglkajhsdlkfjhalsdkf'
txt
'asdkflkjahsdlkjfhlaksglkjdhflkgjhlskdfjhglkajhsdlkfjhalsdkf'
set(txt)
{'a', 'd', 'f', 'g', 'h', 'j', 'k', 'l', 's'}

- 예비학습

list(txt).count("a")
5

- 전체 구현

{k :list(txt).count(k) for k in set(txt)}
{'s': 6, 'l': 9, 'j': 7, 'a': 5, 'h': 7, 'd': 6, 'k': 10, 'f': 6, 'g': 3}

Exercise

import requests
url = 'https://raw.githubusercontent.com/guebin/PP2023/main/posts/01_PythonBasic/Oxford-IIIT.txt'
txt = requests.get(url).content.decode()

위 코드는 이미지 파일명들이 저장된 string을 불러오는 코드들이다.

- 이미지파일이 저장된 형식은 아래와 같다.

  • Abyssinian_1.jpg

  • British_Shorthair_129.jpg

## (1) txt를 적당히 변환하여 이미지 라벨만 표시

sol

temp = txt.split("\n")
temp[0].split("_")[0]
'Abyssinian'
def f(temp):
    return [i.split("_")[0] for i in temp]
lst = f(temp)
lst[::1000]
['Abyssinian',
 'British',
 'Russian',
 'basset',
 'english',
 'keeshond',
 'pug',
 'wheaten']

(2) 강아지와 고양이의 종류는 몇 가지인가?

Sol

- 고양이는 대문자, 강아지는 소문자로 시작한다.

set1 = set(lst)

- 고양이

[s[0].isupper() for s in set1].count(True)
12

- 강아지

[s[0].isupper() for s in set1].count(False)
23

(3) 강아지 종은 dog, 고양이 종은 cat으로 변환

sol

lst2 =["cat"if i[0].isupper() else "dog" for i in lst]
set(lst2)
{'cat', 'dog'}

(4) 강아지와 고양이 사진은 모두 몇장인가?

sol

{s:lst2.count(s) for s in set(lst2)}
{'dog': 4990, 'cat': 2403}
