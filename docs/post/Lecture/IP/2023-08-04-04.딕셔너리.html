Dictionary basic

- 딕셔너리는 key 값과 value로 구성되어있다.

- 순서대로 정리된 자료를 검색할 때는 시퀀스형(리스트, 튜플,string)이 유리하나, 키워드로 검색하고 싶을때는 딕셔너리 타입이 유리하다.

선언

방법 1. 가장 일반적

dct=  {"a" : 1, "b":2}
dct
{'a': 1, 'b': 2}

방법 2. dict

dct = dict(a=1, b=2)
dct
{'a': 1, 'b': 2}

방법 3. 중첩된 리스트

dct = dict([["a",1],["b",2]])
dct
{'a': 1, 'b': 2}

방법 4. 튜플

dct = ("a",1),("b",2)
dict(dct)
{'a': 1, 'b': 2}

원소 추출

- key값을 이용해서 추출해야한다.

dct = dict(dct)
dct
{'a': 1, 'b': 2}
dct["a"]
1

dct1 = {'바나나':0, '사과':1, '오토바이':2, '자동차':3, '자전거':4} 
dct2 = {'과일':['바나나','사과'], '탈것':['오토바이','자동차','자전거']} 
lst = [0,1,0,1,4]

sol

def f(lst) :
    return [k for i in lst for k,v in dct1.items() if v==i ]
f(lst)
['바나나', '사과', '바나나', '사과', '자전거']

Dictionary advanced(\(\star\star\))

Key의 조건

- Key는 중복을 허용하지 않는다.

{"a" : [1,2],"a" : [3,4],"b" : [5,6] }
{'a': [3, 4], 'b': [5, 6]}

- 또한, 리스트, dict의 key, set으로 선언 불가능

dct = {[10,20,30,40]: 'guebin', [10,20,25,40]: 'hanni'} 
dct
TypeError: unhashable type: 'list'
dct = {{0:1}: 'guebin', {1:2}: 'hanni'} 
dct
TypeError: unhashable type: 'dict'
dct = {{'샌드위치','딸기우유'}:'점심', {'불고기','된장찌개','김','콩자반'}: '저녁'}
dct
TypeError: unhashable type: 'set'

- 튜플은 가능하다

dct = {(10,20,30,40): 'guebin', (10,20,25,40): 'hanni'} 
dct
{(10, 20, 30, 40): 'guebin', (10, 20, 25, 40): 'hanni'}
dct[10,20,30,40]
'guebin'
