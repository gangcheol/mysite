예제 2.1 선형추세모형의 적합 예제

library(tidyverse)
library(TTR)
library(lmtest)
library(astsa)
setwd("C:\\Users\\lee\\Desktop\\고급시계열 분석\\제5판_시계열분석_프로그램\\제5판 시계열분석 data")
population <- scan("population.txt")
options(repr.plot.res=200,repr.plot.width=10)
plot.ts(population)
  • 시도표를 그려본 결과 뚜렷한 증가 추세가 보인다.
  • 또한 미세하게 곡선의 형태를 가지므로 1차 선형추세모형과 2차 추세모형을 각각 적합해보기로 한다.
t <- 1:length(population)
t2 <- t^2
fit1 <- lm(population~t)
fit2 <- lm(population~t+t2)
summary(fit1); summary(fit2)

예제 2.2 추세성분과 계절성분을 동시에 갖는 모형 적합 예제

depart <- scan("depart.txt")
depart <- ts(depart, start =c(1984,1),frequency = 12)
plot(depart)

  • 시도표를 그려본결과 추세성분과 계절성분이 관측되었으며, 시간의 흐름에 따라 변동폭이 커짐을 알 수 있다.

  • 따라서 로그변환으로 변동폭을 균등하게 변환

log_depart <- log(depart)
plot(log_depart)
  • 변동폭이 안정화 되었으며 지시함수를 사용한 모형을 적합시킨다.
  • 또한 모수들 사이의 제약조건으로 절편이 없는 모형을 적합시킴
trend <- time(log_depart)-1984 ## 추세성분
month <- factor(cycle(log_depart)) ## 계절성분
reg <- lm(log_depart~0+trend+month) ## 절편이 없는 모형을 생성
summary(reg)
