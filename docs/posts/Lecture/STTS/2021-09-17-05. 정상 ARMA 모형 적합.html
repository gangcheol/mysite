05. 정상 ARMA 모형 적합

Author

GC

Published

September 17, 2021

0. 정상 ARAM 모형의 적합 절차

1. 정상성 판단 : 비정상시계열의 경우 변환, 분해법, 차분 등을 통해 정상시계열로 변환

2. 모형의 식별 : \(ARMA(p.q)\) 차수 선택

3. 모형의 추정 : 우리가 설정한 모델의 대하여 모수를 추정하는 것이다.

4. 모형의 진단 : 화이트 노이즈라는 것에 대한 진단이 필요하다(잔차 분석). 화이트 노이즈 가정에 적합하지 않으면 다시 모형의 식별 단계로 돌아간다.

5.예측모형 : 최종적으로 채택

1. 모형 식별 (model identification)

1. 모형을 적합하기 위한 전처리 단계라고 생각하자.

step .1 정상화 : 분산 안정화변환(이분산성 제거) -> 분해법을 이용하거나 차분, 계절 차분 시행(SACF를 그려 확인)

  • 과대차분을 하지 않도록 주의하여야 한다.

  • 과대차분을 확인하기 위해선 차분된 시계열의 분산과 이전 단계의 분산을 비교 \(Var(\nabla Z) > Var(Z)?\) 좌변이 더 클경우 과대차분의 가능성이 보인다.

  • 샘플사이즈가 적을 경우 \(SACF\)의 도표를 보고 추세가 존재하는지 판단하기 힘들다.

  • 추세나 계절성이 실제로 존재하는지 판단하기 위해 Unit root test(단위 근 검정) 결과를 이용하여,

example \[AR(1) \quad H_0 : \phi =1 \quad vs \quad H_1 :\phi \neq 1\]

* 귀무가설을 기각시 정상성을 만족한다고 볼 수 있다.
  • 계절형 자료가 아닌 경우 대부분 차분의 차수는 0, 1, 2이면 충분하고, 계절형 자료의 경우는 계절의 주기와 같은 값의 차분이면 충분하다.

step.2: ARMA 모형의 차수 결정

library(IRdisplay)
display_png(file="요약표.png",width=700,height=200)
  • AR(p)의 경우

\[H_0 : \phi_{kk} =0 \,vs\, H_1 : \phi_{kk}\neq0 \to |\hat {\phi}_{kk}|>1.96\,/\,\sqrt n \approx 2 / \sqrt n\quad ( k\geq p+1)\]

  • MA(q)의 경우

\[H_0 : \rho_{k} =0 \,vs\, H_1 : \rho_{k}\neq0 \to |\hat {\rho}_{k}|>1.96\,\times\,S.E(\hat {\rho}_k)\quad ( k\geq p+1)\]

\[\begin{eqnarray*}Var\,(\hat {\rho}_k) = \frac 1 n (1+2\rho_1^2+\dots 2p_q^2) = \frac 1n \end{eqnarray*}\]

화이트 노이즈를 가정하기 때문에 \(Var\,(\hat {\rho}_k) =\frac 1n\)이 성립한다. (그냥 그렇다고만)

  • ARMA(p,q)의 경우 : 여러 가지 후보모형을 생각해서 다 적합시킨다.( 모형 선택 기준은 아래와 같다.)

1. : \(AIC\)(Akaike’s Information Criterion, Akaike)

\[ AIC = n\ln(\hat {\sigma}^2)+2(p+q)\]

2. : \(SBC\) (Schwartz’s Bayesian Criterion, Schwartz (1978))

\[ SBC = n\ln(\hat {\sigma}^2) +(p+q)\ln(n)\]

  • 즉, \(AIC, SBC, FPE, BIC, CAT\) 등이 최소가 되도록 모형 선택

step 3.상수항 포함 여부

  • 시계열을 차분한 경우, 상수항 \(\delta\)를 모형에 포함시킬지의 여부를 결정

    • 이 말은 \(E(\nabla Z_t) = 0\)으로 할지 말지에 대한 결정이다.
    • 즉 $ $으로 할지 말지 결정하는 것이다.

  • 차분된 시계열 \(W_t = (1-B)^dZ_t\)의 표본평균과 근사표준오차를 이용하여 \(H_0: \delta = 0\)의 유의성 검정을 시행

  • 검정 통계량 : \(\begin {eqnarray*}\frac {\overline W}{S_{\overline W}} \end{eqnarray*}\)

  • \(S_{\overline W} : \overline {W}_t\)의 표본 분산

  • \(\begin{eqnarray*}\overline W = \frac 1n \sum W_t, S_{\overline W}= \left [\frac {\hat {\gamma}_0}{n}(1+2\hat{\rho}_1+\dots 2\hat{\rho}_k)\right]^{1/2} \end{eqnarray*}\) 단, \(k\)는 유의한 \(SACF\)의 개수

  • \(\hat {\gamma}_0 :\) \(W_t\) 의 표본분산

  • 위에는 실제로 계산하는게 아니니 깊게 생각하지 말자…

2. 모수의 추정 (Parameter Esitmation)

  • 모형을 식별 후 식별한 모형의 모수, 즉 특성을 파악하는 단계이다.

  • 적률추정법, 최소제곱법(CLSE, ULSE), 최대가능도함수를 이용하여 추정

  • 적률추정법의 경우 최종적인 추정방법으로는 추정할수는 없고, 다른 추정방법의 초기치로 사용됨

2-1. 적률추정법 AR(1)

\[Z_t = \phi(Z_{t-1}-\mu)+\varepsilon_t \quad \varepsilon_t \sim WN(0,\sigma^2)\]

  • \(\mu = E(Z_t) \to \hat {\mu}_{MME} = \overline Z\)

  • \(\begin {eqnarray*}Cov(Z_t,Z_{t-1}) = \gamma_1 = \phi\gamma_0 \to \hat {\phi}_{MME} = \frac {\hat {\gamma}_1}{\hat {\gamma}_0} \end{eqnarray*}\)

  • \(Var(Z_t) = \phi^2Var(Z_{t-1}) +\sigma^2 \to \gamma_0 = \phi^2 \gamma_0 + \sigma^2\)

\(\therefore \quad \hat {\sigma}^2_{MME} = \hat {\gamma}_0 (1-\hat{\phi}^2)\)

2-2. 적률추정법 MA(1)

\[Z_t -\mu = \varepsilon_t - \theta \varepsilon_{t-1}\]

  • \(\hat {\mu}_{MME} = \overline Z\)

  • \(\begin{eqnarray*} \hat {\theta}_{MME} = \frac {-1 \pm \sqrt{1-4\hat {\rho}_1^2}}{2\hat{\rho}_1}\end{eqnarray*}\)

    \(if \quad \hat {\rho}_1 = \pm \,0.5 \to \hat {\theta} = \pm 1\) 이므로 가역성을 만족하지 않아 추정량으로 쓸 수 없음

    \(eles \,if \quad |\hat {\rho}_1| > \pm \,0.5 \to \hat {\theta}\) 는 허수로 존재하지 않음

    \(eles \,if \quad |\hat {\rho}_1| < \pm \,0.5 \to |\hat {\theta}|< 1\) 인 경우, 즉 가역성을 만족하는 경우를 찾아서 사용

  • \(\begin {eqnarray*} \hat {\sigma}^2 = \frac {\hat {\gamma_0}}{1+\theta^2}\end{eqnarray*}\)

조건부, 비조건부 최소제곱법과 MLE 추정법은 강의노트 참고

3. 잔차분석 (Residual Analysis)

  • 잔차를 이용하여 \(\varepsilon_t \sim WN(0,\sigma^2)\) 인지를 검정

  • 잔차를 이용한 WN TEST( Portmanteau test )

    • \(H_0: \rho(\varepsilon_1) = \dots \rho_k(\varepsilon_t) = 0\)
    • 위를 검정하며 “box-pierce test”, “Ljung-Box test” 가 있다,
    • 후자의 경우 샘플사이즈가 작은 경우 검정통계량 값을 크게하기 위하여 사용한다.

  • “Jarque and Bera”의 정규성검정

\[\begin {eqnarray*} JB = \frac n6 \left( \frac 1n\sum \frac {\varepsilon_t^3}{\hat \sigma^2} \right )+\left( \frac 1{24}\sum \frac {\varepsilon_t^4}{\hat \sigma^2}-3 \right ) \approx \chi^2(2)\end{eqnarray*}\]

  • 좌항을 왜도, 우항을 첨도로 하여 왜도와 첨도를 가지고 정규성 검정을 시행하는 것이다.

4. 과대적합 진단 (Overfitting diagostics)

  • 모형에 모수를 추가해나가며 과댁적합 진단을 시행

  • 다음과 같은 경우 기존의 잠정모형을 새로운 모형으로 대체한다.

1 잠정모형에 추가된 모수가 유의하다고 판정되는 경우

2 잠정모형에서의 모수 추정값이 과대적합시킨 후 얻어지는 모수 추정값과 큰 차이가 있는 경우

3 과대적합된 모형에서 잔차들의 분산이 잠정모형에서의 잔차들의 분산보다 작아진 경우

5. 참고 : 단위근 검정

\(\Phi(B) = (1-\phi_1 B -\dots - \phi_pB)\)

\[H_0: \phi=1\quad vs \quad H_1 : |\phi| <1\]

  • 차분이 필요한 지 아닌지 검정한다.

  • \(1-B\)에서 \(B=1\) 인 경우 단위근이라고 하는데 이걸 가지는 경우 비정상 시계열이므로 \(ARIMA\) 모형을 적합해야함.

  • 단위근이 있는 경우가 $ $인 경우이다.

  • 차분을 통해 정상 확률 과정으로 변환 가능

  • R 에서 “Dickey -Fuller”로 단위근 검정을 시행 (ex AR(1))

    • case 1 : 적합할 모형에 절편(상수항)이 없는 경우 (nc) 평균이 0인 경우
    • case 2 : 적합할 모형에 절편이 존재하는 경우 (c) 평균이 0이 아닌 경우
    • case 3 : 적합할 모형에 절편과 추세(\(\beta_t\))가 존재하는 경우 (ct) 추세가 있어보이는 경우
