Class 분류 함수 구현

def label_func(fname) :
    if fname[0].isupper() :
      return "cat"
    else : 
      return "dog"
label_func("Egyptian_Mau_120.jpg")
'cat'
label_func("egyptian_Mau_120.jpg")
'dog'
dls = ImageDataLoaders.from_name_func(path,files,label_func,item_tfms=Resize(224))
  • path 경로에서 files에 해당하는 파일들을 불러와서 \(X\)를 만들고 정의한 함수 label_func를 적용해 item_tfms에 정의된 방식으로 \(X\)를 변환하여 저장한다.
dls.show_batch(max_n=8) ## 8개의 그림만보자

cnn_learner

- 1차 목표 : 이미지 파일을 받으면 -> 개인지 고양이인지 판단하는 모형을 만들자

- 2차 목표 : 그 모형에 새로운 이미지 파일을 전달하여 이미지를 분류할 것이다.

- cnn_learner 라는 함수를 이용해서 1차 목표와 2차목표를 달성할 object를 만들것임.
