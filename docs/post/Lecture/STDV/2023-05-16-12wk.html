- folium.Choropleth의 사용법

  • 2개의 데이터를 연결해야함 : 하나는 일반 데이터 프레임, 하나는 json에서 나온 dict
  • 데이터 연결을 위해서는 공유가능한 연결의 매체가 필요하다.
  • 연결의 매개체는 df-key, local_dct2[”],local_dict2['features'][?]['properties']['name']을 이용했다.
  • 연결의 매개체를 name 이아닌 다른 것들도 이용할 수 있다.
