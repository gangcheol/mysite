이미지자료의 이해 및 시각화

- 이미지 자료는 rgb값이 저장된 3개의 매트릭스를 쌓은것

  • shape : (가로픽셀, 세로픽셀, 3)

  • 3은 RGB의 빛의 밝기를 정의

  • 밝기는 0-255사이의 int, 0-1사이의 float으로 정함.

  • int 255 혹은 float 1은 해당색이 매우 밝다는 것을 의미

- 예시 1

r = np.array([0]*25*3).reshape(5,5,3) 
g = np.array([0]*25*3).reshape(5,5,3) 
b = np.array([0]*25*3).reshape(5,5,3) 
r[:3,:3,0] = 255   
g[:3,2:,1] = 255
b[2:,:,2] = 255 
plt.imshow(r+g+b)
<matplotlib.image.AxesImage at 0x1f28bf34590>

- RGB에 작은 숫자를 주면 점점 어두워진다.

r[:3,:3,0] = 80
g[:3,2:,1] = 80
b[2:,:,2] = 80
plt.imshow((r+g+b))

\[l(x) = w_0 + w_{1}x, \quad a(x) = \frac {1}{1+e^{-x}}\]

\(x \in [-5,5]\)의 범위에서 \(a(l(x))\)의 그래프를 그려라. (단 이때 \(w_0=1, w_1=1\))로 설정한다.

def l(x) :
    return w0+w1*x
def a(x) :
    return 1/(1+np.exp(-x))
x = np.linspace(-5,5,1000)
w0=1
w1=1
plt.plot(x,a(l(x)))

ex2

아래와 같은 규칙으로 x,y를 변형해라

\[\bf{xx}_{i}= (cos \,\theta)x_i - (sin \,\theta)y_i\]

\[\bf{yy}_{i}= (sin \,\theta)x_i + (cos \,\theta)y_i\]

\[\theta = \pi/4\]

- 풀이 1

t = np.pi/4

xx = np.cos(t)*x2 - np.sin(t)*y2
yy = np.sin(t)*x2 + np.cos(t)*y2

plt.plot(x2,y2,"o",label=r"(x,y)")
plt.plot(xx,yy,"o",label=r"(xx,yy)")
plt.legend()
fig.tight_layout()

- 풀이 2

\[\begin{bmatrix} \bf {xx} \\ \bf{yy} \end{bmatrix}= \begin{bmatrix} cos\,\theta & -sin\,\theta \\ sin \theta & cos \theta \end{bmatrix} \begin{bmatrix} x_1 \\ y_1 \end{bmatrix}\]

R = np.array([[np.cos(t), -np.sin(t)],[np.sin(t),np.cos(t)]])

xy = np.stack([x2,y2])

xx, yy = R@xy
plt.plot(x2,y2,"o",label=r"(x,y)")
plt.plot(xx,yy,"o",label = r"(xx,yy)")
plt.legend()
<matplotlib.legend.Legend at 0x1f2eba7d090>

ex3

위 과정을 8번 연속진행한 결과를 시각화

plt.legend()
