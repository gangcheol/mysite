2

usapass.txt 는 미국 월별 비행기 승객 수(단위 : 천 명)의 시계열자료이다. log 변환 후 아래의 분석을 수행하시오.

1)

왜 log 변환이 필요한지에 대해 간단히 설명하여라.

Solution

data2 <- scan("usapass.txt")
  • 시도표를 보았을 때 추세성분, 계절성분, 불규칙성분이 모두 관측되었다.
  • 또한 시간의 흐름에 따라 변동폭이 커지는 이분산성을 보이기 때문에 로그변환과 같은 방법을 이용하여 접근해야한다.
  • 실제로 변환시 기존보다 안정된 변동폭을 보여주었다.
par(mfrow=c(1,2))
log_d <- log(data2)
plot.ts(data2,main="변환 전");plot(log_d,type="l",main="변환 후")

2)

적절한 추세모형 적합 후 잔차분석을 하여라.

Solution

t <- 1:length(log_d)
fit3 <- lm(log_d~t)
par(mfrow=c(1,2))
plot(fit3,1);plot(fit3,2)
dwtest(fit3,alternative="two.sided")

  • 잔차분석결과 빨간색 잔차 추세선이 곡선의 형태로 띄어 2차항 등의 추가적인 요소가 필요하다고 판단된다.

  • 또한 몇몇 이상치들이 관측되었다.

  • 그러나 qqplot을 통해 자세히 정규성을 확인한 결과 잔차의 분포가 정규분포에 가까운 형태임을 알 수 있다.

  • 오차항간의 자기상관성 검토 결과 오차항간의 양의 자기상관성이 존재함을 보였다.

3)

적절한 평활법을 적용한 후 잔차분석을 하여라.

Solution

  • 로그변환을 통해 생성된 데이터가 어느 정도 등분산임을 보이고, 계절성분과 추세성분이 관측되었기 때문에 앞선 문제와 동일하게 Winters의 계절지수평활법을 사용하겠다.
data2_ts <- ts(log_d,frequency = 12)
fit4 <- HoltWinters(data2_ts,seasonal = "additive")
ts.plot(resid(fit4), ylab="residual", 
        main="가법모형의 예측오차",lwd=2); abline(h=c(0,0.1,-0.1),lwd=2,col="red",lty="dashed")
  • 잔차의 변동폭이 시간의 흐름에 따라 어느 정도 일정함을 보인다.

4)

  • 적절한 분해법에 의해 각 성분을 분해하여 시계열 그림을 그려라.

  • 추세모형에 의한 분해의 경우 오차항간의 자기상관성 검토 결과 양의 자기상관성을 보였다,
  • 따라서 모델 적합시 자기회귀오차모형(Autoregressive Error Model)을 고려해 볼 수 있다.

4

\(\varepsilon_t\sim_{iid}N(0,1)\) 을 따를 때, 다음의 확률 과정을 따르는 표본의 크기가 100인 시계열을 생성한 후, 각 모의실험 자료에 대하여 다음에 답하여라. 단 AR 혹은 ARMA 과정에서 AR 부분의 차수가 p인 경우 필요한 초기값은 해당하는 확률과정의 평균 𝜇로 놓기로 하자.

모델 1: \(Z_t = \varepsilon_t +0.7\varepsilon_{t-1}\sim MA(1)\quad\theta = -0.7\)

모델 2: \(Z_t - 0.8Z_{t-1}=\varepsilon_{t}\sim AR(1) \quad \phi=0.8\)

모델 3 : \(Z_t -0.5Z_{t-1}= \varepsilon_t+0.3\varepsilon_{t-1} \sim ARMA(1,1) \quad \theta= -0.3,\,\phi=0.5,\,\delta=0\)

  • 앞 선 과제에서 모델 2와 모델 3은 각각 AR(1),ARMA(1,1) 모형임을 보였다.

  • 모형 1의 경우 (\(\theta=-0.7\))

$(B)_t = (B)_t, (B)=1,(B)=1-B=1+0.7B $

(\(\theta=-0.7\))

MA(1)모형을 따른다.

1)

{\(Z_t\)}의 시계열 그림을 그려라.

Solution

z1 <- arima.sim(n=100,list(order=c(0,0,1),ma=0.7),rand.gen=rnorm)
z2 <- arima.sim(n=100,model=list(order=c(1,0,0),ar=0.8),rand.gen = rnorm)
z3 <- arima.sim(n=100,model=list(order=c(1,0,1),ar=0.5,ma=0.3),rand.gen = rnorm)

par(mfrow=c(1,3))
ts.plot(z1,main=(expression(MA(1)~~~theta==-0.7)))
ts.plot(z2,main=(expression(AR(1)~~~phi==0.8)))
ts.plot(z3,main=(expression(ARMA(1,1)~~~ phi==0.5~~~ theta==-0.3)))

(2)

SACF \(\hat {\rho}_k\)와 SPACF \(\hat {\phi}_{kk}\quad (k=0,1,2\dots10)\)의 상관도표를 그려라.

Solution

options(repr.plot.res=250,repr.plot.width=8,repr.plot.hegiht=8)

(3)

2)의 상관도표가 각 확률과정에 대한 ACF, PACF의 이론적인 형태와 비슷한지 논하라.

Solution

  • 빨간점이 이론적인 값, 검은선이 실제로 계산된 값

모델 1 : MA(1)

k=1:10
par(mfrow=c(1,2))
ACF1 <- c(1,0.7/(1+0.7^2),rep(0,9))
PACF1 <-{(-(-0.7)^k)*(1-0.7^2)}/{1-(-0.7^(2*(k+1)))}

z1_acf <- acf(z1,plot=F)
plot(z1_acf[0:10])
points(x=0:10,y=ACF1,col="red",lwd=2)
z1_pacf <- pacf(z1,plot=F)
plot(z1_pacf[0:10])
points(x=1:10,y=PACF1,col="red",lwd=2)
  • ACF의 경우는 이론적으로 시차 1 이후 부터는 0이 나와야하나 그렇지 않다.
  • PACF의 경우도 이론적인 값과는 약간의 차이가 있음을 보인다.

모델 2 : AR(1)

par(mfrow=c(1,2))
k=1:10
ACF2 <- c(1,0.8^k)
PACF2 <- c(0.8,rep(0,9))
z2_acf <- acf(z2,plot=F)
plot(z2_acf[0:10])
points(x=0:10,y=ACF2,col="red",lwd=2)
z2_pacf <- pacf(z2,plot=F)
plot(z2_pacf[0:10])
points(x=1:10,y=PACF2,col="red",lwd=2)
  • ACF와 PACF 둘다 이론적인 값에서 많이 벗어나 있다.

모델 3: ARMA(1,1)

par(mfrow=c(1,2))

  • ACF, PACF 값이 이론적인 값에서 많이 벗어났다.
  • ARMA(1,1) 모형은 PACF 계산값이 복잡해 시차 2까지만 값을 표시하였다.
